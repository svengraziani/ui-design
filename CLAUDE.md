# UI Design Skills

This project contains UI design knowledge as Claude Code skills. Always apply the relevant skill when working on UI code.

## Skill Routing

| Task | Skill |
|------|-------|
| Spacing, padding, margin, layout, whitespace, shadows, hierarchy, responsive design | `ui-design-principles` |
| Picking colors, hex/HSL values, color scales, Tailwind theme, CSS custom properties, semantic colors | `color-palettes` |
| Choosing font-family, typeface, font pairing, font stack, Google Fonts, weights | `font-recommendations` |
| Building a specific component (button, form, table, card, modal, nav, alert, dropdown, etc.) | `component-gallery` |
| Accessibility, a11y, WCAG, ARIA, screen reader, keyboard nav, focus, contrast, alt text, target size | `wcag-2.2` |

When multiple skills apply (e.g. "build a card with good colors"), load all relevant skills.
